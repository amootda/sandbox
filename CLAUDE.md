# CLAUDE.md - 프로젝트 가이드

## 프로젝트 개요

이 프로젝트는 **샌드박스(Sandbox)** 입니다.
만들고 싶은 것들을 자유롭게 실험하고 구현하는 공간입니다.

## 프로젝트 구조 원칙

```
sandbox/
├── package.json          # 프로젝트 메타데이터
├── README.md            # 프로젝트 설명
├── CLAUDE.md            # 이 파일 (작업 참고용)
├── LICENSE              # 라이센스
└── [프로젝트명]/        # 각 아이디어별 디렉토리
    ├── ...
    └── ...
```

## 작업 방식

1. **새로운 아이디어/프로젝트 시작 시:**
   - 프로젝트명으로 새 디렉토리 생성
   - 해당 디렉토리 안에 모든 관련 파일 구성

2. **독립적인 구조:**
   - 각 디렉토리는 독립적인 미니 프로젝트
   - 필요시 각 디렉토리에 자체 package.json, README 등 포함 가능

3. **자유로운 실험:**
   - 다양한 기술 스택 시도 가능
   - 프로토타입, 학습, 테스트 등 모든 용도 환영
